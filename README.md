# AI-Archival-Assistant-System

## 🖥️ 프로그램 소개
전자정부프레임워크를 기반으로 개발된 기록물 보조 시스템. 
OpenAI의 Whisper 모델을 사용하여 자동으로 오디오나 영상 기록물을 요약해주고, 기록물 내용의 태그를 추출하며 찾고 싶은 부분의 타임 스탬프를 반환한다.
